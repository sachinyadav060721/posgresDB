# Advanced SQL in PostgreSQL

This section covers advanced SQL concepts and features in PostgreSQL. Mastering these will help you write powerful, efficient, and maintainable queries.

## Topics to Learn

1. **Window Functions**
   - `ROW_NUMBER()`, `RANK()`, `DENSE_RANK()`, `LEAD()`, `LAG()`, etc.
   - Use cases: running totals, moving averages, ranking, etc.

2. **Common Table Expressions (CTEs)**
   - Syntax: `WITH ... AS (...)`
   - Recursive CTEs
   - Use cases: breaking complex queries, hierarchical data

3. **Advanced Joins and Set Operations**
   - FULL OUTER JOIN, CROSS JOIN
   - INTERSECT, EXCEPT

4. **Subqueries and Correlated Subqueries**
   - In SELECT, FROM, WHERE clauses
   - Performance considerations

5. **Upserts (ON CONFLICT)**
   - `INSERT ... ON CONFLICT ... DO UPDATE`
   - Handling duplicates
